# Employee Attendance System (MERN + Tailwind)

## Overview
Full-stack Employee Attendance System with two roles: Employee & Manager.
Tech: React (Vite) + Tailwind CSS, Node.js, Express, MongoDB (Mongoose).

## Quick start

### Backend
1. cd backend
2. Set `.env` and set `MONGO_URI` and `JWT_SECRET`
3. npm install
4. npm run seed
5. npm start

API will run on http://localhost:5000

### Frontend
1. cd frontend
2. npm install
3. create `.env` with `REACT_APP_API_URL=http://localhost:5000/api`
4. npm run start

Seeded accounts:
- Manager: alice.manager@example.com / password123
- Employee: bob.employee@example.com / password123
